Definition 5.10 Dense sets
✓
#

Lean declarations

A set is dense if its closure is the entire ambient space. That is,

\[ \overline A = X. \]

Definition 5.12 Separable spaces
✓
#

Lean declarations

  • LeanTopology.ClosureInterior.IsSeparable_4_13

A topological space is separable if it contains a countable dense subset. Equivalently,

\[ \exists D\subseteq X \quad \text{such that} \quad D \text{ is countable} \quad \text{and} \quad \overline D = X. \]
